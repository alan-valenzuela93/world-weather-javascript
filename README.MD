### Requirements:

Nodejs >= v12   

### Modules to install:

- Webpack
- Webpack-cli
- style-loader
- css-loader
- file-loader

To have a development server, run on terminal:

```
npm run dev
```


To make deployment files, run on terminal:

```
npm run build
```

This will generate a folder called 'dist' with the files needed to deploy.
